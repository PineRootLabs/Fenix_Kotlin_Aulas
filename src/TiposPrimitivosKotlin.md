# Tipos Primitivos em KotlinEsta tabela resume os tipos primitivos disponíveis em Kotlin, juntamente com suas características principais.| Família  | Tipo Primitivo | Classe Envolucro | Tamanho (bits)   | Exemplo         ||----------|----------------|-------------------|------------------|-----------------|| Inteiros | `Byte`         | `kotlin.Byte`     | 8                | `10`            ||          | `Short`        | `kotlin.Short`    | 16               | `1000`          ||          | `Int`          | `kotlin.Int`      | 32               | `100000`        ||          | `Long`         | `kotlin.Long`     | 64               | `10000000000L`  || Reais    | `Float`        | `kotlin.Float`    | 32               | `3.14f`         ||          | `Double`       | `kotlin.Double`   | 64               | `3.14159`       || Lógico   | `Boolean`      | `kotlin.Boolean`  | *Dependente*     | `true`, `false` || Literais | `Char`         | `kotlin.Char`     | 16               | `'A'`, `'@'`    |**Observações:*** O tamanho do `Boolean` não é fixo e depende da JVM (Java Virtual Machine) e da forma como é utilizado. Geralmente, ocupa o menor espaço endereçável, que pode variar.* Em Kotlin, todos esses tipos são tratados como objetos, mas o compilador frequentemente otimiza o uso dos tipos primitivos subjacentes para melhor performance. As "Classes Envolucro" são usadas quando é necessário um objeto real, por exemplo, em coleções que não podem armazenar tipos primitivos diretamente.
